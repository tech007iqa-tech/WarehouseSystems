Since you need live marketplace values, you can use highly resilient, open-source crawling tools that scrape pricing without getting blocked by anti-bot walls.

Crawlee (by Apify)
What it is: A powerful, open-source web scraping and browser automation library built for Node.js.

Why it’s great for your warehouse: Marketplaces like eBay use heavy JavaScript and anti-bot fingerprints. Crawlee automatically manages proxy rotation, mimics human browser behavior, and handles headless browsers (like Chromium and Playwright) out of the box.

How to use it: You can write a script that takes your list of laptop model numbers, queries e-commerce search pages, and extracts the current "Buy It Now" or "Sold" prices into a clean JSON format.
